Fink - Документация

Это комплект различных документов, составленных для Fink. Некоторые из них могут быть также полезны тем, кто использует Mac OS X или Darwin без Fink и хочет узнать о переносе ПО Unix.

Документация пользователя

Текущая документация пользователя Fink:

  • Руководство пользователя Fink - инсталляция самого Fink, инсталляция пакетов, обновление до новых версий Fink. Содержит инструкции относительно исходного кода и бинарной версии.
  • Fink Advanced Topics Guide - covers more advanced concepts than those covered in the User's Guide.
  • Выполнение X11 в Darwin и Mac OS X - концепции, инсталляция и запуск (также предназначается для пользователей Darwin и Mac OS Xl в целом)

Комплект документов, более полных, но отчасти устаревших и более не поддерживаемых:

  • Инсталляция и обновление - инсталляция Fink и обновление до новой версии
  • Использование - использование Fink и инсталлированного ПО
  • Fink ReadMe - файл ReadMe для дистрибуции на основе исходного кода
  • Доступ CVS - доступ к хранилищу Fink CVS для получения последних исходных пакетов между выпусками версий

Документация разработчика

  • Пособие по политике безопасности - обязательное пособие для изучения теми, кто поддерживает пакеты в Fink или хотел бы добавить собственные пакеты.
  • Использование Fink: пособие разработчика (файл 2MB pdf) - начиная от презентации на Конференции O'Reilly Mac OS X (также возможной как файл PowerPoint )
  • Подсказки по переносу - заметки о переносе приложений Unix в Darwin
  • Пособие по пакетированию - дополнение к учебнику пакетирования, которое дает реальные примеры и дает вводную информацию о пакетировании для начинающих.
  • Учебник пакетирования - о создании и поддержке пакетов Fink
  • The Fink Developer Wiki - includes developer-related material that is under construction.

Другие документы

  • Руководство по интернационализации - материал о развитии деятельности по интернационализации веб-сайтов
  • Сетевой этикет списков рассылки - как наилучшим образом применять списки рассылки Fink
