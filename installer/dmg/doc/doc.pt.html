Fink - Documentação

Este é um conjunto de vários documentos escritos para o Fink. Alguns destes documentos também podem ser úteis a pessoas que usam o Mac OS X sem o Fink mas que desejam aprender a portar software Unix.

Documentação do usuário

Documentação do usuário atualizada:

  • Guia do usuário do Fink - cobre a instalação do Fink, instalação de pacotes e atualização para uma nova versão do Fink. Contém instruções para ambas as versões binária e código fonte.
  • Guia de tópicos avançados do Fink - cobre conceitos mais avançados do que os cobertos no Guia do Usuário. (em inglês)
  • Executando o X11 no Mac OS X - cobre conceitos, instalação e execução (também direcionado a usuários do Mac OS X em geral). (em inglês)

Alguns documentos que são mais completos mas estão um pouco desatualizados e não são mais mantidos:

  • Instalação e atualização - como instalar o Fink ou atualizá-lo para uma versão nova (em inglês)
  • Formas de uso - como usar o Fink e os softwares instalados (em inglês)
  • LeiaMe do Fink - o LeiaMe da distribuição de códigos fontes (em inglês)
  • Acesso via CVS - como acessar o repositório CVS do Fink para obter os fontes dos pacotes mais recentes antes da liberação de uma nova versão (em inglês)

Documentação do desenvolvedor

  • Manual de política de segurança - leitura obrigatória para todos os que mantêm pacotes no Fink ou gostariam de adicionar seus próprios pacotes. (em inglês)
  • Using Fink: A Developer's How To (arquivo PDF com 2MB) - slides de uma apresentação na O'Reilly Mac OS X Conference (também disponível como um arquivo PowerPoint) (em inglês)
  • Dicas para portar software - notas para os interessados em portar aplicações Unix para o Mac OS X (em inglês)
  • Tutorial de empacotamento - um complemento ao Manual de empacotamento que foca exemplos reais e faz uma introdução sobre empacotamento para iniciantes. (em inglês)
  • Manual de empacotamento - como criar e manter pacotes no Fink (em inglês)
  • The Fink Developer Wiki - material relacionado a desenvolvimento que ainda está em construção. (em inglês)

Outros documentos

  • Guia de internacionalização - material sobre o esforço em andamento de internacionalização do site (em inglês)
  • Netiqueta para as listas de discussão - como melhor usar as listas de discussão do Fink. (em inglês)
